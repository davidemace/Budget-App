# Mace Budget Worker

A Cloudflare Worker-based budget management application.

## Project Structure

```
mace-budget-worker/
├── package.json
├── wrangler.toml
├── README.md
├── migrations/
├── src/
├── styles/
└── views/
```

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start development server:
   ```
   npm run dev
   ```

3. Deploy:
   ```
   npm run deploy
   ```

## Features

- Dashboard overview
- Budget management
- Credit card tracking
- Paycheck management
- Mortgage tracking
- Financial goals
- Money management services
- Smart recommendations
